清单-安卓APP

原则

  • 功能足够，体检小
  • 权限少，少泄露隐私
  • 更可能是开源软件
  • 使用 googleplay 和 F-Droid更新应用，apkpure/apkmirror 替代，尽量不使用国产APP

日常

  • 微信
  • 支付宝

启动器

  • Unlauncher - 试了一圈这个最简单
  • Olauncher 每日切换壁纸
  • Before Launcher

笔记、清单

  • WorkFlowy
  • Microsoft To Do 微软平台同步

效率

  • Loop Habit Tracker 习惯养成记录
  • Cx File Explorer 文件管理：免费，支持FTP/webdav/网盘映射
  • 输入法：谷歌拼音，旧但简单 ｜ Fcitx 安卓版，还有词库

编辑器

  • Markor : markdown编辑器

阅读

  • KOReader 静读天下 Moon+ Reader 、fbReader : 开源，中文字体支持稍逊
  • Tachiyomi ：漫画神器

浏览器

  • Firefox
  • Via : 体积小功能足够

音乐

  • Musicolet Music Player - 功能全面，对播放列表支持很好
  • Listen 1 音乐播放器：在线音乐汇集
  • Clean Music Player：类似 1by1 的文件夹播放，缺点是好久不更新后台播放有问题

视频

  • VLC : 支持多种格式播放
  • bilibili：谷歌版无广告

Podcast

  • AntennaPod : 开源播放器

RSS

  • Feeder

网盘、同步

  • 坚果云 : 国产优质
  • Syncthing : 全平台同步用，可以用 Syncthing-Fork | F-Droid

图片

  • Snapseed

外语学习

  • AnkiDroid - 记忆神器
  • MDict : 配合词典文件

其他

  • AirVisual : 空气质量查询、预测
  • 时钟插件：Sony Clock widgets Retro Clock Widget
  • 屏幕时钟 com.by_syk.screenclock：非常小巧精致
